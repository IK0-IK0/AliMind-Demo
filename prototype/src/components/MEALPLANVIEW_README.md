# MealPlanView Component

A React component for displaying generated meal plans with day-by-day breakdowns, recipe cards, and detailed recipe information.

## Overview

The `MealPlanView` component provides a comprehensive interface for viewing meal plans generated by the `mealPlanService`. It displays:

- Meal plan summary with duration and calorie targets
- Day-by-day breakdown with daily nutrition totals
- Recipe cards for each meal with nutrition information
- Detailed recipe view dialog with ingredients and instructions
- "Generate New Plan" functionality

## Features

- **Loading State**: Displays loading spinner and skeleton loaders during meal plan generation
- **Session-Only Demo Alert**: Clearly indicates that plans are not saved
- **Day-by-Day Organization**: Groups meals by day with daily nutrition summaries
- **Color-Coded Meal Types**: Visual distinction between breakfast, lunch, dinner, and snacks
- **Recipe Cards**: Compact cards showing key recipe information
- **Detailed Recipe Dialog**: Full recipe details including ingredients, instructions, prep/cook time, and macros
- **Responsive Design**: Works on mobile, tablet, and desktop devices
- **Material-UI Styling**: Consistent with the green/teal health-focused design system

## Usage

```typescript
import { MealPlanView } from './components/MealPlanView';
import { generateMealPlan, MealPlanConstraints } from './services/mealPlanService';

function MyComponent() {
  const [mealPlan, setMealPlan] = useState(() => {
    const constraints: MealPlanConstraints = {
      targetCalories: 2000,
      dietaryRestrictions: [],
      maxPrepTime: 60,
      budget: 'medium',
      difficulty: 'easy',
      equipment: 'basic'
    };
    return generateMealPlan(constraints, 1);
  });

  const handleGenerateNew = () => {
    const constraints: MealPlanConstraints = {
      targetCalories: 2000,
      dietaryRestrictions: [],
      maxPrepTime: 60,
      budget: 'medium',
      difficulty: 'easy',
      equipment: 'basic'
    };
    setMealPlan(generateMealPlan(constraints, 1));
  };

  return (
    <MealPlanView 
      mealPlan={mealPlan} 
      onGenerateNew={handleGenerateNew}
    />
  );
}
```

## Props

### `mealPlan: MealPlan` (required)

The meal plan object to display. Must include:
- `durationDays`: Number of days in the plan
- `totalDailyCalories`: Target daily calories
- `items`: Array of meal plan items

### `onGenerateNew: () => void` (required)

Callback function invoked when the "Generate New Plan" button is clicked. Should generate a new meal plan and update the component's state.

### `isLoading?: boolean` (optional)

Controls the loading state of the component. When `true`, displays:
- A centered loading spinner (CircularProgress)
- "Generating Your Meal Plan..." message
- Skeleton loaders showing the structure of the meal plan
- Hides the actual meal plan content

Default: `false`

**Example with loading state:**

```typescript

  const [isLoading, setIsLoading] = useState(false);
```

## Component Structure

### Meal Plan Summary

Displays:
- Duration (number of days)
- Daily calorie target
- Total number of meals
- Average meals per day
- "Generate New Plan" button

### Day Breakdown

For each day:
- Day number heading
- Daily nutrition summary (calories, protein, carbs, fat)
- Grid of meal cards

### Meal Cards

Each card shows:
- Meal type chip (color-coded)
- Recipe name
- Recipe description (truncated)
- Calories and macros (P/C/F)
- Total time (prep + cook)
- "View Recipe" button

### Recipe Detail Dialog

Full recipe information:
- Recipe name and meal type
- Description
- Prep time, cook time, difficulty, calories
- Macronutrient breakdown
- Ingredients list
- Step-by-step instructions

- **Mobile (xs)**: Single column layout, full-width cards

## Notes

- **Session-Only**: Plans are NOT persisted to a database. This is a demo component for displaying generated plans.
- **No Editing**: The component is read-only. Users cannot edit meals or recipes.
- **No Saving**: There is no "Save Plan" functionality. Plans exist only in memory.
- **Generate New**: The "Generate New Plan" button creates a completely new plan with potentially different recipes.
